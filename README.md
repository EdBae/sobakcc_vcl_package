# sobakcc_vcl_package
sobakcc vcl package for DELPHI

[VCL] Sobakcc Controls Package for D7

개인적으로 사용하기 위하여 만든 컨트롤들로 FlatStyle의 컴포넌트가 대부분이다.

델파이 7에서 팩키지 등록함.
